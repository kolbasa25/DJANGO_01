# Пульт охраны банка

## Как запустить скрипт

Скрипт подключается к удалённой базе данных и выводит:
- общее количество пропусков
- количество активных пропусков

### 1. Клонируйте репозиторий

```bash
git clone <repo_url>
cd <repo_folder>
```

### 2. Создайте виртуальное окружение

```bash
python -m venv venv
```

Активируйте окружение.

```bash
source venv/bin/activate
```

### 3. Установите зависимости

```bash
pip install -r requirements.txt
```

### 4. Создайте файл `.env`

В корне проекта создайте файл `.env` и добавьте в него параметры подключения к базе данных:

```env
DB_HOST=хост
DB_PORT=порт
DB_NAME=название базы
DB_USER=пользователь
DB_PASSWORD=пароль
```

### 5. Запустите скрипт

```bash
python main.py
```

### Ожидаемый результат

```text
Всего пропусков 100
Активных пропусков 89
```
